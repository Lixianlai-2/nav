#### Parcel 的复用

1. 添加 package.json`npm init -y`
2. 处理无法 build 的问题
   1. 删除了 main.js 就 OK 了
3. 重新设置 scripts

```javascript
"scripts": {
    "start": "parcel src/index.html -p 3000 --open",
    "build:parcel": "parcel build ./src/index.html --public-url ./ --no-source-maps",
    "build": "npm run clean && npm run build:parcel",
    "clean": "rm -rf dist/*"
  }
```
